15/**
16 * @fileoverview Conversions from plain string to goog.html types for use in
17 * legacy APIs that do not use goog.html types.
18 *
19 * This file provides conversions to create values of goog.html types from plain
20 * strings. These conversions are intended for use in legacy APIs that consume
21 * HTML in the form of plain string types, but whose implementations use
22 * goog.html types internally (and expose such types in an augmented, HTML-type-
23 * safe API).
24 *
25 * IMPORTANT: No new code should use the conversion functions in this file.
26 *
27 * The conversion functions in this file are guarded with global flag
28 * (goog.html.legacyconversions.ALLOW_LEGACY_CONVERSIONS). If set to false, it
29 * effectively "locks in" an entire application to only use HTML-type-safe APIs.
30 *
31 * Intended use of the functions in this file are as follows:
32 *
33 * Many Closure and application-specific classes expose methods that consume
34 * values that in the class' implementation are forwarded to DOM APIs that can
35 * result in security vulnerabilities. For example, goog.ui.Dialog's setContent
36 * method consumes a string that is assigned to an element's innerHTML property;
37 * if this string contains untrusted (attacker-controlled) data, this can result
38 * in a cross-site-scripting vulnerability.
39 *
40 * Widgets such as goog.ui.Dialog are being augmented to expose safe APIs
41 * expressed in terms of goog.html types. For instance, goog.ui.Dialog has a
42 * method setSafeHtmlContent that consumes an object of type goog.html.SafeHtml,
43 * a type whose contract guarantees that its value is safe to use in HTML
44 * context, i.e. can be safely assigned to .innerHTML. An application that only
45 * uses this API is forced to only supply values of this type, i.e. values that
46 * are safe.
47 *
48 * However, the legacy method setContent cannot (for the time being) be removed
49 * from goog.ui.Dialog, due to a large number of existing callers. The
50 * implementation of goog.ui.Dialog has been refactored to use
51 * goog.html.SafeHtml throughout. This in turn requires that the value consumed
52 * by its setContent method is converted to goog.html.SafeHtml in an unchecked
53 * conversion. The conversion function is provided by this file:
54 * goog.html.legacyconversions.safeHtmlFromString.
55 *
56 * Note that the semantics of the conversions in goog.html.legacyconversions are
57 * very different from the ones provided by goog.html.uncheckedconversions: The
58 * latter are for use in code where it has been established through manual
59 * security review that the value produced by a piece of code must always
60 * satisfy the SafeHtml contract (e.g., the output of a secure HTML sanitizer).
61 * In uses of goog.html.legacyconversions, this guarantee is not given -- the
62 * value in question originates in unreviewed legacy code and there is no
63 * guarantee that it satisfies the SafeHtml contract.
64 *
65 * To establish correctness with confidence, application code should be
66 * refactored to use SafeHtml instead of plain string to represent HTML markup,
67 * and to use goog.html-typed APIs (e.g., goog.ui.Dialog#setSafeHtmlContent
68 * instead of goog.ui.Dialog#setContent).
69 *
70 * To prevent introduction of new vulnerabilities, application owners can
71 * effectively disable unsafe legacy APIs by compiling with the define
72 * goog.html.legacyconversions.ALLOW_LEGACY_CONVERSIONS set to false. When
73 * set, this define causes the conversion methods in this file to
74 * unconditionally throw an exception.
75 *
76 * Note that new code should always be compiled with
77 * ALLOW_LEGACY_CONVERSIONS=false. At some future point, the default for this
78 * define may change to false.
79 */

100/**
101 * Performs an "unchecked conversion" from string to SafeHtml for legacy API
102 * purposes.
103 *
104 * Unchecked conversion will not proceed if ALLOW_LEGACY_CONVERSIONS is false,
105 * and instead this function unconditionally throws an exception.
106 *
107 * @param {string} html A string to be converted to SafeHtml.
108 * @return {!goog.html.SafeHtml} The value of html, wrapped in a SafeHtml
109 * object.
110 */
111goog.html.legacyconversions.safeHtmlFromString = function(html) {
112 goog.html.legacyconversions.throwIfConversionsDisallowed();
113 return goog.html.SafeHtml.createSafeHtmlSecurityPrivateDoNotAccessOrElse(
114 html, null /* dir */);
115};

189/**
190 * Throws an exception if ALLOW_LEGACY_CONVERSIONS is false. This is useful
191 * for legacy APIs which consume HTML in the form of plain string types, but
192 * do not provide an alternative HTML-type-safe API.
193 */
194goog.html.legacyconversions.throwIfConversionsDisallowed = function() {
195 if (!goog.html.legacyconversions.ALLOW_LEGACY_CONVERSIONS) {
196 throw Error(
197 'Error: Legacy conversion from string to goog.html types is disabled');
198 }
199 goog.html.legacyconversions.reportCallback_();
