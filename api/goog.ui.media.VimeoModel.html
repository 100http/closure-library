Class goog.ui.media.VimeoModel

Extends
  • goog.ui.media.MediaModel

The goog.ui.media.Vimeo media data model. It stores a required videoId field, sets the vimeo URL, and allows a few optional parameters.

new VimeoModel( videoId, opt_caption, opt_description, opt_autoplay )

Parameters
videoIdstring

The vimeo video id.

opt_caption(string|undefined)

An optional caption of the vimeo video.

opt_description(string|undefined)

An optional description of the vimeo video.

opt_autoplay(boolean|undefined)

Whether to autoplay video.
