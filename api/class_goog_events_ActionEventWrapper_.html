goog.events.EventWrapper

Event wrapper for action handling. Fires when an element is activated either by clicking it or by focusing it and pressing Enter.

Instance Methods

listen(target, listener, opt_capt, opt_scope, opt_eventHandler)code »

Adds an event listener using the wrapper on a DOM Node or an object that has implemented goog.events.EventTarget. A listener can only be added once to an object.

Specified by: goog.events.EventWrapper
