goog.soy.REQUIRE_STRICT_AUTOESCAPE → boolean

Whether to require all Soy templates to be "strict html". Soy templates that use strict autoescaping forbid noAutoescape along with many dangerous directives, and return a runtime type SanitizedContent that marks them as safe.

If this flag is enabled, Soy templates will fail to render if a template returns plain text -- indicating it is a non-strict template.
