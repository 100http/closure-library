class BasicStat

final

Tracks basic statistics over a specified time interval.

Statistics are kept in a fixed number of slots, each representing an equal portion of the time interval.

Most methods optionally allow passing in the current time, so that higher level stats can synchronize operations on multiple child objects. Under normal usage, the default of goog.now() should be sufficient.
