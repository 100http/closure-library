class FlatButtonRenderer

goog.ui.ControlRenderer
  └ goog.ui.ButtonRenderer
      └ goog.ui.FlatButtonRenderer

Flat renderer for goog.ui.Buttons. Flat buttons can contain almost arbitrary HTML content, will flow like inline elements, but can be styled like block-level elements.

Returns true if this renderer can decorate the element. Overrides goog.ui.ButtonRenderer#canDecorate by returning true if the element is a DIV, false otherwise.

Takes an existing element and decorates it with the flat button control. Initializes the control's ID, content, tooltip, value, and state based on the ID of the element, its child nodes, and its CSS classes, respectively. Returns the element. Overrides goog.ui.ButtonRenderer#decorate.
