enum SliderBase.EventType

Typestring

Event types used to listen for dragging events. Note that extent drag events are also sent for single-thumb sliders, since the one thumb controls both value and extent together; in this case, they can simply be ignored.

Values and Descriptions

DRAG_END

User is done dragging a thumb

DRAG_EXTENT_END

User is done dragging the extent thumb

DRAG_EXTENT_START

User started dragging the extent thumb

DRAG_START

User started dragging a thumb

DRAG_VALUE_END

User is done dragging the value thumb

DRAG_VALUE_START

User started dragging the value thumb
