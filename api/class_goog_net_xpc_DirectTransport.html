A direct window to window method transport.

If the windows are in the same security context, this transport calls directly into the other window without using any additional mechanism. This is mainly used in scenarios where you want to optionally use a cross domain transport in cross security context situations, or optionally use a direct transport in same security context situations.

Note: Global properties are exported by using this transport. One to communicate with the other window by, currently crosswindowmessaging.channel, and by using goog.getUid on window, currently closure_uid_[0-9]+.
