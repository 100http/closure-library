JsonStreamParser.Options → {allowCompactJsonArrayFormat: (boolean|undefined), deliverMessageAsRawString: (boolean|undefined)}

Configuration spec for newly created JSON stream parser:

allowCompactJsonArrayFormat: whether allows compact JSON array format, where null is represented as empty string, e.g. "[1, ,2]".

deliverMessageAsRawString: whether to deliver the raw message string without decoding into JS object. Semantically insignificant whitespaces in the input may be kept or ignored.
