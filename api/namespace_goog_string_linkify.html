linkifyPlainText(text, opt_attributes, opt_preserveNewlines)code »

deprecated

Takes a string of plain text and linkifies URLs and email addresses. For a URL (unless opt_attributes is specified), the target of the link will be _blank and it will have a rel=nofollow attribute applied to it so that links created by linkify will not be of interest to search engines.

Deprecated: Use goog.string.linkify.linkifyPlainTextAsHtml instead.

Parameters
textstring

Plain text.

opt_attributesObject<string, ?(string|number|goog.string.TypedString|Object<string, ?(goog.string.Const|string)>)>=

Attributes to add to all links created. Default are rel=nofollow and target=_blank. To clear those default attributes set rel='' and target=''.

opt_preserveNewlinesboolean=

Whether to preserve newlines with <br>.

Returns
string

HTML Linkified HTML text. Any text that is not part of a link will be HTML-escaped.
