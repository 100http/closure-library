Interface goog.history.Html5History.TokenTransformer

Provided By
  • goog.history.Html5History.TokenTransformer

A token transformer that can create a URL from a history token. This is used by goog.history.Html5History to create URL when storing token without the hash fragment.

Given a window.location object containing the location created by createUrl, the token transformer allows retrieval of the token back via retrieveToken.
