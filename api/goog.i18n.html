currencyCodeMap → Object<string, string>

Deprecated, this data is not being updated. Please use goog.i18n.currency.

The mapping of currency symbol through intl currency code. The source of information is mostly from wikipedia and CLDR. Since there is no authoritative source, items are judged by personal perception.

If an application need currency support that available in tier2, it should extend currencyCodeMap to include tier2 data by doing this: goog.object.extend(goog.i18n.currencyCodeMap, goog.i18n.currencyCodeMapTier2);

warning Deprecated

Use goog.i18n.currency.getLocalCurrencyPattern instead.
