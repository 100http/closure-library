A string-like object that carries a content-type and a content direction.

IMPORTANT! Do not create these directly, nor instantiate the subclasses. Instead, use a trusted, centrally reviewed library as endorsed by your team to generate these objects. Otherwise, you risk accidentally creating SanitizedContent that is attacker-controlled and gets evaluated unescaped in templates.
