Enumeration goog.db.IndexedDb.EventType

Provided By
  • goog.db.IndexedDb
Typestring

Event types fired by a database.

VIEW SOURCE

Values

ABORT

Fired when a transaction is aborted and the event bubbles to its database.

CLOSE

Fired when the database connection is forcibly closed by the browser, without an explicit call to IDBDatabase#close. This behavior is not in the spec yet but will be added since it is necessary, see https://www.w3.org/Bugs/Public/show_bug.cgi?id=22540.

ERROR

Fired when a transaction has an error.

VERSION_CHANGE

Fired when someone (possibly in another window) is attempting to modify the structure of the database. Since a change can only be made when there are no active database connections, this usually means that the database should be closed so that the other client can make its changes.
