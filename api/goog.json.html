parse( s ) → (Object|null)

Parses a JSON string and returns the result. This throws an exception if the string is an invalid JSON string.

Note that this is very slow on large strings. If you trust the source of the string then you should use unsafeParse instead.

Parameters
s*

The JSON string to parse.

(Object|null)
