class Uri.QueryData

finalstruct

Class used to represent URI query parameters. It is essentially a hash of name-value pairs, though a name can be present more than once.

Has the same interface as the collections in goog.structs.

new Uri.QueryData(opt_query, opt_uri, opt_ignoreCase)

Parameters
opt_query?string=

Optional encoded query string to parse into the object.

opt_uri?goog.Uri=

Optional uri object that should have its cache invalidated when this object updates. Deprecated -- this is no longer required.

opt_ignoreCaseboolean=

If true, ignore the case of the parameter name in #get.
