Module goog.net.rpc.HttpCors

Module ID
  • goog.net.rpc.HttpCors

Provides CORS support for HTTP based RPC requests.

As part of net.rpc package, CORS features provided by this class depend on the server support. Please check related specs to decide how to enable any of the features provided by this class.
