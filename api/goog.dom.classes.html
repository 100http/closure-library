addRemove( element, classesToRemove, classesToAdd ) → void

Adds zero or more classes to an element and removes zero or more as a single operation. Unlike calling goog.dom.classes.add and goog.dom.classes.remove separately, this is more efficient as it only parses the class property once.

If a class is in both the remove and add lists, it will be added. Thus, you can use this instead of goog.dom.classes.swap when you have more than two class names that you want to swap.

warning Deprecated

Use goog.dom.classlist.addRemove instead.

Parameters
element(Node|null)

DOM node to swap classes on.

classesToRemove(string|Array<string>|null)

Class or classes to remove, if null no classes are removed.

classesToAdd(string|Array<string>|null)

Class or classes to add, if null no classes are added.
