Module goog.i18n.dateIntervalSymbols

Module ID
  • goog.i18n.dateIntervalSymbols

Date interval formatting symbols for all locales.

File generated from CLDR ver. 31.0.1

To reduce the file size (which may cause issues in some JS developing environments), this file will only contain locales that are frequently used by web applications. This is defined as proto/closure_locales_data.txt and will change (most likely addition) over time. Rest of the data can be found in another file named "dateintervalsymbolsext.js", which will be generated at the same time together with this file.
