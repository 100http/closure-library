class MediaModel

An base data value class for all media data models.

MediaModels are exact matches to the fields defined in the Yahoo RSS media specification http://search.yahoo.com/mrss/.

The current common data shared by medias is to have URLs, mime types, captions, descriptions, thumbnails and players. Some of these may not be available, or applications may not want to render them, so null values are allowed. goog.ui.media.MediaRenderer checks whether the values are available before creating DOMs for them.

Types

MediaModel.Category

A taxonomy to be set that gives an indication of the type of media content, and its particular contents.

MediaModel.Credit

Indicates an entity that has contributed to a media object.

MediaModel.Medium

Supported mediums, found here: http://video.search.yahoo.com/mrss

MediaModel.MimeType

The supported media mime types, a subset of the media types found here: http://www.iana.org/assignments/media-types/ and here http://en.wikipedia.org/wiki/Internet_media_type

MediaModel.Player

Constructs a player containing details of the player's URL and optionally its size.

MediaModel.SubTitle

A reference to the subtitle URI for a media object.

MediaModel.Thumbnail

Constructs a thumbnail containing details of the thumbnail's image URL and optionally its size.
