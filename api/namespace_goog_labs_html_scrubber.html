scrub(tagWhitelist, attrWhitelist, html)code »

Replaces tags not on the white-list with empty text nodes, dropping all attributes, and drops other non-text nodes such as comments.

Parameters
tagWhitelistObject<string, boolean>

a set of lower-case tag names following the convention established by goog.object.createSet.

attrWhitelistObject<string, Object<string, function(?(string|goog.html.SafeHtml|goog.html.SafeStyle|goog.html.SafeUrl)): ?(string|goog.html.SafeHtml|goog.html.SafeStyle|goog.html.SafeUrl)>>

maps lower-case tag names and the special string "*" to functions from decoded attribute values to sanitized values or null to indicate that the attribute is not allowed with that value.

For example, if attrWhitelist['a']['href'] is defined then it is used to sanitize the value of the link's URL.

If attrWhitelist['*']['id'] is defined, and attrWhitelist['div']['id'] is not, then the former is used to sanitize any id attribute on a <div> element.
