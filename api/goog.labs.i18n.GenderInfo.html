Class goog.labs.i18n.GenderInfo

GenderInfo provides a method to determine the gender of a list/array of objects when one knows the gender of each item of the list. It does this in a locale sensitive manner. The constructor initializes the object based on the locale data from the current goog.labs.i18n.ListFormatSymbols.
