Renders an emoji palette.

new EmojiPaletteRenderer(defaultImgUrl)

Parameters
defaultImgUrl?string

Url of the img that should be used to fill up the cells in the emoji table, to prevent jittering. Will be stretched to the emoji cell size. A good image is a transparent dot.
