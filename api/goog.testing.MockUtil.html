getAllProperties(obj)

Get all properties of an object.

Depending on the browser this will return both enumerable and non-enumerable properties, or only enumerable properties. It will not, however, get the properties on the Object.prototype since some tests depend on things like toString() to not be mocked if it has not been overridden.
