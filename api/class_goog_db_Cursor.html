next(opt_key)code »

Advances the cursor to the next position along its direction. When new data is available, the NEW_DATA event will be fired. If the cursor has reached the end of the range it will fire the COMPLETE event. If opt_key is specified it will advance to the key it matches in its direction.

This wraps the native #continue method on the underlying object.

Parameters
opt_key(number|string|Date|Array<?>)=

The optional key to advance to.
