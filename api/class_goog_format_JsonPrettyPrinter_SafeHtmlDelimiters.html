class JsonPrettyPrinter.SafeHtmlDelimiters

goog.format.JsonPrettyPrinter.TextDelimiters
  └ goog.format.JsonPrettyPrinter.SafeHtmlDelimiters

A container for the delimiting characters used to display the JSON string to an HTML <pre> or <code> element. It escapes the names and values before they are added to the output. Use this class together with goog.format.JsonPrettyPrinter#formatSafeHtml.
