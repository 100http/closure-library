class FlashObject

goog.Disposable
  └ goog.events.EventTarget
      └ goog.ui.Component
          └ goog.ui.media.FlashObject
All implemented interfaces:
goog.disposable.IDisposable, goog.events.Listenable

A very simple flash wrapper, that allows you to create flash object programmatically, instead of embedding your own HTML. It extends goog.ui.Component, which makes it very easy to be embedded on the page.

new FlashObject(flashUrl, opt_domHelper)

Parameters
flashUrl(string|goog.html.TrustedResourceUrl)

The Flash SWF URL. If possible pass a TrustedResourceUrl. string is supported for backwards-compatibility only, uses goog.html.legacyconversions, and will be sanitized with goog.html.SafeUrl.sanitize() before being used.

opt_domHelper?goog.dom.DomHelper=

An optional DomHelper.
