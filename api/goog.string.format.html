Namespace goog.string.format

Performs sprintf-like conversion, i.e. puts the values in a template. DO NOT use it instead of built-in conversions in simple cases such as 'Cost: %.2f' as it would introduce unnecessary latency opposed to 'Cost: ' + cost.toFixed(2).
