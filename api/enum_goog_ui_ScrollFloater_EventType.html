enum ScrollFloater.EventType

Typestring

Events dispatched by this component.

Values and Descriptions

DOCK

Dispatched when the component returns to its original state. The event is cancellable.

FLOAT

Dispatched when the component starts floating. The event is cancellable.

PIN

Dispatched when the component gets pinned to the bottom of the container element. This event is cancellable.
