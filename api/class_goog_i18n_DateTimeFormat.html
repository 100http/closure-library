Format the given date object according to preset pattern and current locale.

Parameters
date(Date|goog.date.Date)

The Date object that is being formatted.

opt_timeZone?goog.i18n.TimeZone=

optional, if specified, time related fields will be formatted based on its setting. When this field is not specified, "undefined" will be pass around and those function that really need time zone service will create a default one.

Returns
string

Formatted string for the given date. Throws an error if the date is null or if one tries to format a date-only object (for instance goog.date.Date) using a pattern with time fields.
