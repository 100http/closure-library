this.createDom<T>( tagName, opt_attributes, ...var_args ) → R

Returns a dom node with a set of attributes. This function accepts varargs for subsequent nodes to be added. Subsequent nodes will be added to the first node as childNodes.

So: createDom(goog.dom.TagName.DIV, null, createDom(goog.dom.TagName.P), createDom(goog.dom.TagName.P)); would return a div with two child paragraphs

An easy way to move all child nodes of an existing element to a new parent element is: createDom(goog.dom.TagName.DIV, null, oldElement.childNodes); which will remove all child nodes from the old element and add them as child nodes of the new DIV.

Parameters
tagName(string|goog.dom.TagName<(T|null)>)

Tag to create.

opt_attributes(Object|string|null|undefined)

If object, then a map of name-value pairs for attributes. If a string, then this is the className of the new element.

var_args...(Object|string|null)

Further DOM nodes or strings for text nodes. If one of the var_args is an array or NodeList, its elements will be added as childNodes instead.

Returns
R

Reference to a DOM node. The return type is {!Element} if tagName is a string or a more specific type if it is a member of goog.dom.TagName (e.g. {!HTMLAnchorElement} for goog.dom.TagName.A).
