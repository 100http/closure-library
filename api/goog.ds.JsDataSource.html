Data source whose backing is JavaScript data

Names that are reserved for system use and shouldn't be used for data node names: eval, toSource, toString, unwatch, valueOf, watch. Behavior is undefined if these names are used.

new JsDataSource( root, dataName, opt_parent )

Parameters
root(Object|null)

The root JS node.

dataNamestring

The name of this node relative to the parent node.

opt_parent(Object|null|undefined)

Optional parent of this JsDataSource.
