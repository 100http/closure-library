An object for monitoring the saving of files. This emits ProgressEvents of the types listed in goog.fs.FileSaver.EventType.

This should not be instantiated directly. Instead, its subclass goog.fs.FileWriter should be accessed via goog.fs.FileEntry#createWriter.
