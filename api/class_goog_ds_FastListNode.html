setChildNode(key, value)code »

Sets a child node. Creates the child if it does not exist. To set children at a certain index, use a key of the form '[index]'. Note, that you can only set values at existing numeric indices. To add a new node to this list, you have to use the add method.

Calling this function makes any child nodes previously obtained for name invalid. You should not use these child nodes but instead obtain a new instance by calling getChildNode.
