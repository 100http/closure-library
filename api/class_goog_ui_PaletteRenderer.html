class PaletteRenderer

goog.ui.ControlRenderer
  └ goog.ui.PaletteRenderer

Default renderer for goog.ui.Palettes. Renders the palette as an HTML table wrapped in a DIV, with one palette item per cell:

setContent(element, content)code »

Overrides goog.ui.ControlRenderer#setContent for palettes. Locates the HTML table representing the palette grid, and replaces the contents of each cell with a new element from the array of nodes passed as the second argument. If the new content has too many items the table will have more rows added to fit, if there are less items than the table has cells, then the left over cells will be empty.
